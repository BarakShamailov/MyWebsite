# personal-website
## 🌐 Personal Portfolio Website

A modern, responsive personal portfolio website built to showcase my background, projects, and transition into AI Engineering.

## 🚀 Overview

This website serves as my professional landing page, presenting my experience, technical skills, and ongoing journey into AI and backend development.

It includes smooth animations, an interactive UI, and a fully functional contact form powered by EmailJS.

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Animations:** Intersection Observer API + Canvas API
- **Email Service:** EmailJS
- **Styling:** Custom CSS (responsive design)
- **Deployment:** GitHub Pages
---

## ✨ Features

### 🔹 Responsive Design
Fully responsive layout optimized for desktop and mobile devices.

### 🔹 Dynamic Navbar
- Scroll-based styling
- Active section highlighting

### 🔹 Mobile Navigation
- Hamburger menu
- Smooth open/close transitions
- Scroll lock when menu is open

### 🔹 Scroll Reveal Animations
Elements fade in smoothly using the **Intersection Observer API**.

### 🔹 Contact Form (EmailJS)
- Sends emails directly without a backend
- Loading state + success/error feedback
- Fully client-side implementation

### 🔹 Smooth Scrolling
Seamless navigation between sections.

### 🔹 Animated Background
Custom **particles animation** using Canvas:
- Dynamic particle movement
- Interactive connection lines
- Optimized for performance
